Bienvenue sur QR Ludo Generator

Cette application propose de générer différends Qr Code à l'aide de formulaire, voici la liste des Qr Code qui peuvent être construit.

  • Unique

    QR Code de forme simple contenant soit du texte soit un fichier audio.

  • Multiple

    QC Code contenant de multiple Qr code unique

  • Exercice Qr Code

    QR code de type exercice composé d'un QR Code question et de QR Code réponses

  • Exercice de Reconnaissance Vocale

    Cette onglet dispose de sous-onglets permettant de réaliser des exercices de reconnaissance vocale

    • Question Ouverte

      • Création d'un QR Code composé d'une question et d'une réponse

    • QCM

      • Création d'un QR Code composé de sa question et des réponses possible à cette question, ces dernières seront lus sur l'application mobile

  • Serious Game

    Création d'un jeu ludique qui sera composé de plusieurs énigmes auxquelles l'utilisateur de l'application mobile devra répondre
